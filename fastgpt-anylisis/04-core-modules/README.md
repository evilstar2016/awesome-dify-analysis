# FastGPT 核心模块分析文档

## 📚 文档概览

本目录包含 FastGPT 项目各核心模块的深度分析文档，包括详细的功能说明、数据模型、时序图等。

## 📁 模块目录

### 1. [应用管理模块](./app-management/README.md) ⭐ NEW
应用生命周期管理、应用类型、版本控制

**核心内容**:
- 应用类型体系（Simple/Agent/Workflow）
- 应用版本管理
- 工具应用（HTTP/MCP工具集）

### 2. [对话管理模块](./chat-management/README.md) ⭐ NEW
对话会话管理、消息存储、反馈系统

**核心内容**:
- 对话会话与消息存储
- 多来源对话支持（API/分享/第三方）
- 用户反馈系统

### 3. [知识库管理模块](./knowledge-base/README.md)
知识库创建、文档处理、向量化、检索

**核心内容**:
- 知识库数据模型
- 文档处理（8种格式）
- 向量化与检索策略
- 7个核心时序图

### 4. [工作流引擎模块](./workflow/README.md)
工作流编排、节点执行、变量系统

**核心内容**:
- 节点类型详解（15+种）
- 执行引擎机制
- 变量与队列管理
- 交互式工作流

### 5. [Agent智能体模块](./agent/README.md)
Agent应用、工具调用、迭代执行

**核心内容**:
- Agent两种形式
- 工具调用机制
- Function Calling实现
- 多轮迭代执行

### 6. [MCP工具模块](./mcp-tools-plugins/README.md)
MCP协议集成、工具发现、调用

**核心内容**:
- MCP客户端/服务端
- 工具动态发现
- 工具执行引擎

### 7. [Prompt管理模块](./prompt-management/README.md)
Prompt模板、变量替换、版本管理

**核心内容**:
- Prompt类型体系
- 变量替换机制
- 第三方集成分析

### 8. [插件系统模块](./plugin-system/README.md) ⭐ NEW
系统插件、团队安装、标签管理

**核心内容**:
- 系统插件管理
- 团队插件安装
- 插件费用配置
